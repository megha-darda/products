
## Description

It uses mongodb. MongoDb Compass should be downloaded.

## TechStack

Nestjs, MongoDb

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
